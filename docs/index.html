Rust terminal toolkit for Bittime

bittime-cli

Market data, account inspection, spot order workflows, funding history, live WebSocket streams, and MCP tool exposure from one compact command line.

Install View commands

Built for real exchange workflows

Public endpoints, signed private requests, live streams, and automation output are handled consistently.

Market Data

Tickers, prices, book ticker, order book depth, recent trades, aggregate trades, and exchange info.

Private API

Account info, balances, assets, open orders, all orders, trade history, and funding history.

Trading

Place market or limit orders, query order state, cancel active orders, and inspect active books.

WebSocket

Market depth, private order updates, private balance updates, raw channels, gzip frames, and ping/pong.

MCP

Expose CLI commands as JSON-RPC tools for agents and local automation environments.

Safe Credentials

Use CLI flags, environment variables, or a local config file stored with restrictive permissions.

Install

Build directly with Cargo and run from your terminal.

From source

git clone https://github.com/ibidathoillah/bittime-cli.git
cd bittime-cli
cargo install --path .
bittime --help

Live WebSocket streams

The CLI handles Bittime market gzip frames and user stream keepalive, with bounded stream options for scripts and CI smoke tests.

gzipmarket frame decoding
pongmarket and user heartbeats
rawprivate user channels
CIbounded `--ws` smoke mode

Streaming examples

bittime ws depth usdtidr
bittime ws depth usdtidr --limit 1 --seconds 15
bittime ws orders
bittime ws balances
bittime ws user user_order_update

Command surface

Use tables while working interactively and JSON when composing scripts.

Market

bittime market ping
bittime market ticker USDTIDR
bittime market orderbook USDTIDR -l 10
bittime market trades USDTIDR -l 5

Account

bittime account balance
bittime account assets usdt
bittime account trades-v2 USDTIDR
bittime -o json account info

Trade

bittime trade open-orders USDTIDR
bittime trade all-orders USDTIDR
bittime trade book-orders USDTIDR -l 5
bittime trade cancel USDTIDR --order-id 123

Funding

bittime funding withdraw-history --coin usdt
bittime funding deposit-history --coin usdt
bittime funding otc-va-code --bank-id 7
bittime funding otc-withdraw-history --limit 10
